# If 文

> **ソースコード**: [04_if.go](../02_flowcontrol/04_if.go)

## 概要

Go の `if` 文は条件分岐を行います。`for` ループと同様、条件式に括弧は不要です。

## コード解説

```go
func sqrt(x float64) string {
    if x < 0 {
        return sqrt(-x) + "i"
    }
    return fmt.Sprint(math.Sqrt(x))
}
```

- 負の数が渡されたら、虚数として表現（例: `-4` → `2i`）
- 正の数なら平方根を計算

## ポイント

- 条件式を囲む括弧 `()` は不要
- 波括弧 `{}` は必須（1行でも省略不可）
- `for` 文と同じスタイル
